# Bit masks
- Used to access specific bits in a given bytcode, note a shift is still typically needed to extract the appropriate value
## AND bitmask

```c
x:      11001001

y:      00001111

x & y:  00001001
```


## OR bitmask

```c
x:      11110000

y:      00001111

x | y:  11111111
```
