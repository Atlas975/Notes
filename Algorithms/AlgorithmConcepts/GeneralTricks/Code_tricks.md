# Swapping variables 
- Standard temp variable 
```python
temp = a 
a = b
b = temp
```

- Arithmetic methods
```python
a = a + b
b = a - b
a = a - b
```

```python
a = a * b
b = a / b
a = a / b
```

- Bitwise swap with XOR
```python
a = a ^ b 
b = a ^ b
a = a ^ b
```
