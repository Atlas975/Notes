# Algebraic inverse
- Reverse operations
- eg: y=4x -> y=x/4
# Inverse rules
- Each y **MUST** map to unique x to have an inverse as the inverse is also a function.
- So not every function has an inverse
- Example
> f(x)=x^2
y cannot map to both -2 and 2
