## Boolean type 
- Can be defined as
```
let is_bool:bool = true;
```
- Can also result from a logical expression
```
let c = 10 > 2;
```

## Char type 
- Can be defined as 
```
let letter:char = 'e'
```

## String type 
- Can be defined as
```
let str_1:&str = "Rust Programming";
```
