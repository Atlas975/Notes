# Bit masks
- Used to access specific bits in a given bytcode, note a shift is still typically needed in most cases to extract the appropriate value
## AND bitmask

```c
x:      10001101

y:      01010111

x & y:  00000101
```


## OR bitmask

```c
x:      10001101

y:      01010111

x | y:  11011111
```
